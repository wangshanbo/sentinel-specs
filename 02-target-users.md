# 02 · Target Users · 目标用户画像

## 主用户群（按优先级）

### 画像 1：独立创业者 / 单人开发者 ★★★★★

- **背景**：有想法、有少量时间、可能会一点点编程但不专业；预算有限
- **典型场景**：
  - 想做一个 SaaS 落地页 + Stripe 付费墙验证想法
  - 想做一个内容订阅站
  - 想做一个工具站（AI 包装、数据查询、计算器类）
- **痛点**：找外包贵、自己学编程慢、用 SaaS 受限
- **付费意愿**：高（应用本身能赚钱时愿意付月费 ¥99-299）

### 画像 2：内容创作者 / KOL ★★★★

- **背景**：在小红书 / 抖音 / B 站 / 公众号有粉丝，想做"自己的小工具"或落地页
- **典型场景**：
  - 给课程做报名页 + 收款
  - 给社群做入群登记 + 自动审核
  - 给品牌做活动页
- **痛点**：现成工具受平台限制、想要个性化
- **付费意愿**：中（按项目付费，¥199-999/项目）

### 画像 3：小微企业主 ★★★★

- **背景**：餐饮、零售、教培、健身房等，需要简单数字化
- **典型场景**：
  - 餐厅菜单 + 在线点单
  - 课程预约系统
  - 客户档案管理
- **痛点**：买不起定制开发，套模板不合适
- **付费意愿**：中（月费 ¥99-499）

### 画像 4：运营 / PM / 设计师 ★★★

- **背景**：在公司内但需要快速做内部工具或验证想法
- **典型场景**：
  - 内部数据看板
  - 实验性 landing page
  - 临时活动报名
- **痛点**：技术资源紧、自己想动手
- **付费意愿**：中低（个人订阅或公司报销）

### 画像 5：教师 / 培训师 ★★★

- **背景**：高校 / K12 / 职业培训
- **典型场景**：
  - 课程作业提交平台
  - 在线小测验
  - 课程社区
- **痛点**：教育平台贵且束缚多
- **付费意愿**：低-中

## 反画像（明确不服务）

### 反画像 1：专业开发者

- 他们的工具是 Cursor / Claude Code / VS Code / JetBrains
- 我们隐藏代码、不暴露 LSP / 调试 / 终端 — 对他们是负担
- **如果一个产品决定让专业开发者满意，普通人就一定不满意**

### 反画像 2：大企业 IT 部门

- 需要 SSO / SAML / 私有部署 / 合规审计 / SLA
- Phase 1-3 不做，等 C 端跑通再考虑

### 反画像 3：需要实时音视频 / 高并发游戏 / 重 ML 推理的场景

- 沙箱跑不了，部署成本极高
- 不在我们能力范围内

### 反画像 4：完全没有审美 / 完全不知道想要什么的人

- 我们能帮你提炼意图、追问澄清，但不能从「我想做点什么」生造出有意义的应用
- 期望管理：Sentinel 是"想法翻译器"，不是"想法生成器"

## JTBD（Jobs To Be Done）

### Job 1：把脑子里的想法变成可分享的真实应用

- **触发**：朋友吃饭聊天有了 idea / 看到别人产品有了灵感
- **现状方案**：找外包（贵、慢）、自己学（难、慢）、用 SaaS（不灵活）
- **理想方案**：当晚说出来，第二天发链接给朋友试用

### Job 2：拿应用做生意（收钱、收订阅、做服务）

- **触发**：想验证商业模式 / 已有用户希望产品化
- **现状方案**：用 Shopify/Gumroad（受平台规则）、找开发者
- **理想方案**：5 分钟做完接付款，立刻能赚钱

### Job 3：根据用户反馈持续改进

- **触发**：用户说「我希望 X」「这里 bug 了」
- **现状方案**：再花钱让外包改 / 自己折腾代码
- **理想方案**：把反馈丢给 AI，自动改并发布

### Job 4：根据数据优化

- **触发**：「转化率怎么这么低」「用户都在哪一步流失」
- **现状方案**：手动加 Google Analytics + 自己分析 + 自己改
- **理想方案**：AI 直接看到数据 + 提建议 + 直接改

## 用户认知与心智边界

### 我们必须假设用户：

- ❌ 不知道什么是 React / Vue / 数据库 / 后端
- ❌ 不会装 Node.js / npm / Docker
- ❌ 不理解 deploy / domain / SSL 这些术语
- ❌ 看到代码会感到焦虑或恐惧
- ❌ 不会自己 debug
- ❌ 不会写需求文档（需要 AI 帮忙提炼）

### 但我们可以假设用户：

- ✅ 会用浏览器、会点击按钮
- ✅ 见过其他应用（小红书、淘宝、各种 SaaS），有视觉审美直觉
- ✅ 能用自然语言（中/英文）描述「我想做什么」
- ✅ 能识别「这个对/不对」「我喜欢/不喜欢」
- ✅ 愿意为有用的工具付费（前提是真有用）
